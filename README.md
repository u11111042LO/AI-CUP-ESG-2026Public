# AICUP-2026-VeriPromiseESG

Repository for AICUP 2026 VeriPromise ESG experiments, training scripts, summaries, and submission files.

## Structure

- `configs/`: experiment configuration files.
- `scripts/`: training and submission-generation scripts.
- `summaries/`: public leaderboard history and experiment summaries.
- `submissions/`: generated submission CSV files.
- `docs/`: experiment notes and reproduction instructions.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

See [docs/how_to_reproduce.md](docs/how_to_reproduce.md) for reproduction details.
